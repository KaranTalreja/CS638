HAPPYB - Happy Birthday !!!

#adhoc

 


Problem Statement


Alim, Shipu & Sufian are good friends. They are programmers & their coach is Dipu sir. Sometimes  They give training to newbies.  Dipu  sir ordered to create a very easy problem set for new programmers. They think of a very easy problem. This problem uses some condition in a program and print the result. The condition  are,







Condition




Print








If   date(d) 7 and month(m) 1




Happy   Birthday To Shipu.








If   date(d) 29 and    month(m) 4




Happy Birthday   To Dipu Sir.








If   date(d) 30 and month(m) 5




Happy   Birthday To Fahad vai.








If   date(d) 21 and month(m) 6




Happy   Birthday To Sufian.








If   date(d) 11 and month(m) 12




Happy   Birthday To Alim.








If   another day and month




This is   an ordinary day.








Input 


Input is a sequence of lines. Each line contains two integer d ( 1≤ d ≤ 31 ) and m ( 1 ≤  m≤ 12 ).


Output


For each line of input print a single line following by the Print Colum .


Sample Input/Output:








Sample Input




Sample Output








29    4


11    12


30      5
24   6 




Happy Birthday To Dipu Sir.


Happy Birthday To Alim.


Happy Birthday To Fahad vai. 
This is an ordinary day.
